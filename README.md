# Deep Learning for Computer Vision Assignment

This GitHub repository hosts the code and documents related to the Deep Learning for Computer Vision Assignment.

**Repository Link:** [https://github.com/matthewsultana-um/deep-learning-for-computer-vision-assignment-ms/tree/main](https://github.com/matthewsultana-um/deep-learning-for-computer-vision-assignment-ms/tree/main)
**Quiz Link:** [here](https://docs.google.com/forms/d/e/1FAIpQLSfyvSdnDKwu0dCFBvbAi4C78e9BdVWNgvqa4GHU2qIa-UiOow/viewform?usp=dialog)
**Hugging Face Spaces:** (https://huggingface.co/spaces/mattsult-um/ari5118)

For this assignment, the chosen topic is **Non-Maximum Suppression (NMS) and Evaluation Metrics**. The project explores the NMS algorithm, Intersection over Union (IoU) thresholding, Soft-NMS, precision-recall curves, Average Precision (AP), and mean Average Precision (mAP).

## Repository Contents

| File / Directory | Description |
|---|---|
| `README.md` | Contains links related to the academic papers used for the study notes. |
| `study_notes.pdf` | A set of notes used as a guide for learning NMS and Average Precision. |
| `quiz_with_rationale.pdf` | A list of 10 questions related to NMS and AP. The interactive Google Form format of this quiz can be found [here](https://docs.google.com/forms/d/e/1FAIpQLSfyvSdnDKwu0dCFBvbAi4C78e9BdVWNgvqa4GHU2qIa-UiOow/viewform?usp=dialog). |
| `walkthrough.ipynb` | A Jupyter Notebook highlighting the algorithm from first principles. |
| `ai_journal.pdf` | Documentation detailing the AI usage throughout the project. |
| `simulator/` | Contains folders related to the simulator to visualize the concepts of NMS and mAP. It includes a `README.md` with instructions to run locally and a `requirements.txt`. It is also hosted on [Hugging Face Spaces](https://huggingface.co/spaces/mattsult-um/ari5118). |
| `further_reading/` | Contains additional links to the academic papers referenced in the study notes. |
| `slides.pdf` | Presentation slides summarizing this project. |
